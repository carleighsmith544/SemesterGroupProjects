Features

Reserve a Seat: Users can select a specific seat to reserve.

Reserve a Range of Seats: Users can reserve multiple seats in a row at once.

Cancel a Reservation: Users can cancel a reservation for a specific seat.

Cancel a Range of Reservations: Users can cancel multiple reserved seats in a row.

Reset Seating Chart: Users can clear all reservations and reset the seating chart.

Visual Representation: The application displays the seating chart with reserved seats marked in a different color.
